3. Директории - id (int unsigned), directory (int unsigned), name (varchar(255)).

SQL запросы:
1. Список товаров одной директории каталога.
"select * from Товары where directory= id_подкаталога_производителя";
2. Список товаров одной директории каталога, включая все подкаталоги.
"select * from Товары where directory IN (select id from Директории where directory=id_основной_директории)";
Или "select * from Товары where type=type_id".
